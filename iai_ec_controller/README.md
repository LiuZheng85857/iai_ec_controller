# IAI EC电缸控制系统

基于Python的IAI EC电缸控制程序，支持EtherNet/IP协议通信。

## 功能特性

- ✅ EtherNet/IP协议通信
- ✅ 完整的运动控制（原点复位、定位、点动、推压）
- ✅ 参数读写和管理
- ✅ 实时状态监控
- ✅ 图形化控制界面
- ✅ 命令行交互模式
- ✅ 完善的日志记录
- ✅ 模块化设计，易于扩展

## 系统要求

- Python 3.7+
- Windows/Linux/macOS
- 网络连接到EC电缸
